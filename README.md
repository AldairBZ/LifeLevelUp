# 🌱 LifeLevelUp

> "Cada elección saludable que haces... impacta más allá de ti."

LiveLevelUp es una **web app interactiva** que une tu bienestar personal con la salud global del planeta.
Un espacio donde cuidar tu cuerpo, tus hábitos y tu entorno se convierte en un juego de conciencia y evolución.

---

## 🌍 ¿Qué es LifeLevelUp?

Una experiencia dividida en dos secciones conectadas:

### 🧍 Salud y Bienestar Personal
- Crea tu avatar, configura tus hábitos y observa cómo tu esperanza y calidad de vida cambian con cada decisión.

### 🌐 Calidad de Vida Mundial
- Explora el estado del planeta y simula cómo tus acciones (y las de muchos) pueden mejorar o empeorar su salud.

---

## 🧬 ¿Cómo funciona?

### 🩺 Salud y Bienestar
- El usuario se registra indicando **edad y género**.
- Se genera un **avatar vital**, con:
  - Esperanza de vida
  - Estado físico y de bienestar general
- El avatar puede ser equipado con:
  - ⚙️ Hábitos (alimentación, descanso, consumo, etc.)
  - 🏃 Actividades (ejercicio, sedentarismo, movilidad)
- Los equipamientos modifican su estado tomando como base **información médica real** obtenida de fuentes confiables.
- La app muestra en tiempo real:
  - Cambios en salud y esperanza de vida
  - Recomendaciones y mensajes personalizados

### 🌎 Calidad de Vida del Planeta
- Panel visual del estado ambiental global, dividido por zonas y aspectos (aire, agua, biodiversidad…)
- El usuario puede elegir acciones (reciclar, dejar el coche, plantar árboles, etc.)
- Cada acción muestra:
  - Su impacto estimado en el planeta
  - La posibilidad de simular ese impacto si más personas se suman
- El sistema permite configurar:
  - "¿Qué pasaría si 100, 1.000 o un millón de personas hicieran lo mismo?"

> Porque el cambio colectivo empieza con una sola persona: **tú**.

## 📁 Estructura del Proyecto

```text
livelevelup/
├── frontend/         # Interfaz web (Angular/React + Tailwind)
├── backend/          # API y lógica de usuario (Node.js + Express)
├── avatar-system/    # Núcleo de cálculo vital y visualización
├── habit-engine/     # Equipamiento y lógica de efectos por hábitos
├── world-status/     # Panel global ambiental + simulador de acciones
├── data-scraper/     # Obtención dinámica de datos médicos y ecológicos
├── db/               # MongoDB para usuarios, hábitos, acciones
└── assets/           # Iconos, fondos, animaciones y sonidos
```
